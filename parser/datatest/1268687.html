Formelle forudsætninger:A willingness to think creatively.
Perform basic image editing tasks such as cropping, resizing and optimizing images.
The ability to create a flow chart for a website.

The student should understand how to implement a basic website. It is not important that the student understands HTML code, but it is expected that he/she understands the file structure behind a web site. This should include knowledge of how to publish content for the Internet by uploading files to web server.

It is not required that students have programming experience, but students with no programming experience should be prepared to make an extra effort. 
I øvrigt skal man opfylde IT-Universitetets generelle optagelseskrav.
Læringsmål:After completing the course the student should be able to:

Use the tools of Flash, to design interactive content for the Internet.

Identify possibilities and problems that arise when communicating with motion graphics.

Plan and implement an Internet based project, including communicating the ideas and structure with the help of storyboards and flowcharts. 
Fagligt indhold:The course considers the technical and visual capabilities of interaction design and ActionScript, and uses these, among other things, to examine linear and interactive communication.

Introduction to animation theory, motion graphics, interaction and interface design, programmed animation, and basic building blocks for game design.

Through this course students will establish a basic understanding of the fundamentals of ActionScript 3.0 that will serve as a solid foundation for further learning.

Students will also establish an understanding of the technical aspects of publishing interactive projects for the Internet.

The course will provide an overview / status of the development of HTML-5, various tools and issues, including the new HTML-5 features of Flash CS-6. 
Læringsaktiviteter:14 ugers undervisning bestående af forelæsninger og øvelser

Lectures and exercises, reading, and 2 compulsory assignments.
The lecture and exercise period is mixed.

Der er mulighed for øvelseshold og eksamen på både dansk og engelsk.

NB!! Kursusspærring!!
Grundet identisk kursusindhold kan du ikke følge dette kursus, hvis du har bestået enten Multimedieproduktion på internet eller Interaktive multimedier.

NB!! Course Restriction!!
Due to identical course content you may not follow this course if you have already taken either of the courses Multimedia Production for the Internet or Interactive Multimedia
